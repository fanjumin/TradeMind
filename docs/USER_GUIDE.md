## 1. 快速上手

### 安装

```bash
git clone https://github.com/fanjumin/TradeMind.git
cd TradeMind
pip install -r requirements.txt
```

### 启动

```bash
python3 unified_app.py --port 8081
```

### 打开浏览器

访问 http://localhost:8081

### 主界面

顶部导航栏包含所有页面的链接。右侧搜索框可直接搜索股票代码或名称。

**使用方式**:
1. 在输入框输入股票代码
2. 选择分析维度标签页
3. 查看图表和评分结果

## 6. 本地备份

**路由**: `/backup`

自动备份 experience.db（存储用户偏好、自选股、预警配置等）。

### Watchdog 自动备份
- 每 10 秒检测文件变化
- 变化时自动创建备份文件
- 备份文件存储在 `~/.trademind-sync/backups/`

### 手动操作
- **立即备份**: 点击「立即备份」按钮
- **下载**: 下载备份文件到本地
- **回退**: 恢复到指定备份版本
- **删除**: 删除不需要的备份
- **上传**: 上传外部备份文件

## 7. 企业微信推送

在 `/backup` 页面配置:

1. 点击推送开关 → **启用**
2. 点击 **测试** 按钮 → 企业微信收到测试消息
3. Watchdog 自动备份时 → 自动推送通知

推送链路:
```
experience.db 变化 → Watchdog → 本地备份 → POST /api/alert/send
  → alert_push.send_wecom_message() → 企业微信 API → 用户收到消息
```

> 注意: 需要 TradeMind 服务正常运行（端口 8081），接收人在 `alert_push.py` 中配置（默认: FanJuMin）。

## 8. 版本管理

**路由**: `/update`

### 检查更新
查看当前版本号和远程最新版本

### 下载源码
从 GitHub 下载最新版本源码

### 更新
执行 `git pull` 拉取最新代码

### 回退
切换到指定历史版本

### 安全机制
- 更新前自动备份配置文件
- `git stash` 保护本地修改
- 回退不影响 `~/.trademind-sync/` 备份目录

## 9. 常见问题

### Q: 启动时报端口被占用
```bash
# 修改端口
python3 unified_app.py --port 8082
```

### Q: 股票数据不显示
- 检查网络连接（数据源: Sina Finance）
- A股交易时段才有实时数据
- 非交易日显示前一交易日数据

### Q: 推送测试失败
- 确认端口 8081 服务正常运行
- 检查 easykai.cn:8081/send 是否可达
- 查看 `~/.trademind-sync/config/push.json` 中的错误日志

### Q: 如何迁移备份到新电脑
```bash
# 旧电脑
cp ~/.trademind-sync/backups/*.db /U盘/

# 新电脑
# 在 /backup 页面使用上传功能导入
```

### Q: 如何恢复出厂设置
```bash
# 删除配置文件（注意先备份）
rm -rf ~/.trademind-sync/
```
