第三部分: 差距分级
======================================================================

🔴 P0 严重 (架构性缺失):
1. REST API — 无对外接口, 无法集成 (修复难度: 低, 1-2天)
2. 实时扫描引擎 — 仅手动触发, 无自动发现 (修复难度: 中, 1-2周)
3. 策略市场 — 无策略复用/分享 (修复难度: 中)
4. 实盘交易 — 分析与执行断层 (修复难度: 高, 可先模拟)

🟡 P1 中等 (功能深度不足):
5. 财务数据 — 缺三表/利润表/现金流 (baostock可补)
6. 多资产 — 缺ETF/可转债/指数 (架构扩展)
7. 因子研究 — 缺IC/IR/分层回测 (新增模块)
8. 行业产业链 — 缺上下游/景气度 (数据获取难)

🟢 P2 轻度 (体验优化):
9. 移动端 10. 社区社交 11. 多因子模型 12. Level2数据
13. 宏观政策 14. 多周期联动 15. 异常大单监控
