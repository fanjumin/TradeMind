使用教程

从获取 API Key 到完成第一次调用，10分钟入门。

1 获取 API Key

点击首页「开始使用」或直接访问 /subscribe。 免费版点击即可生成 Key，Premium 需微信支付订阅。

2 第一次 API 调用

用 curl 测试你的 Key 是否生效：

curl -H "X-API-Key: 你的_key" https://easykai.cn/api/v2/stock/600519/quote

成功返回包含实时行情的 JSON。

3 Python 调用示例 
import requests

API_KEY = "tm-your-key-here"
BASE = "https://easykai.cn/api/v2"

headers = {"X-API-Key": API_KEY}

# 获取股票行情
r = requests.get(f"{BASE}/stock/600519/quote", headers=headers)
data = r.json()
print(f"当前价: {data['data']['stock']['price']}")

# 获取回测结果
r = requests.get(f"{BASE}/backtest/600519?strategy=macd", headers=headers)
print(r.json())
4 系统集成

将 TradeMind API 集成到你的交易系统、分析面板或微信机器人中：

  1. 定时调用 /api/v2/scan 获取市场信号
  2. 为自选股调用 /api/v2/stock/{symbol}/full 获取深度分析
  3. 用 /api/v2/ask/{symbol} 获取 AI 研判

注意事项

  • Free 版每天 100 次调用，Premium 10000 次
  • 数据仅供参考，不构成投资建议
  • LLM 分析存在一定延迟（~5-10秒）
